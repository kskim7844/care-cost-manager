# 요양비 세부관리 시스템

## 개요
기존 요양관리 시스템의 총액 입력을 보완하는 **세부 내역 관리 프로그램**입니다.

## 기능
- **5개 항목** 세부 관리: 진료약제비 / 의료소모품 / 계약의사 / 가정간호 / 기타비용
- 수급자(입소자) ID 기반 항목별 관리
- 날짜/수급자/월별 **조회/검색**
- 월별 합계 **대시보드**
- **엑셀 내보내기** (시스템 업로드용 / 세부내역용)
- SQLite DB (대용량 대응)

## 실행 방법

### 1. 사전 준비 (최초 1회)
```bash
npm install
```

### 2. 서버 실행
```bash
node server.js
```

### 3. 브라우저 접속
```
http://localhost:3000
```

## 클라우드 배포 (선택)
- **Railway, Render, Fly.io** 등에 그대로 배포 가능
- DB 파일: `care_cost.db` (자동 생성)

## 디렉토리 구조
```
care-cost-manager/
├── server.js          # Express + SQLite 백엔드
├── package.json
├── care_cost.db       # SQLite DB (자동 생성)
└── public/
    ├── index.html     # 프론트엔드 메인
    ├── style.css      # 스타일
    └── app.js         # 클라이언트 로직
```

## 시스템 업로드용 엑셀
- 상단 **[시스템 업로드용 엑셀]** 버튼 클릭
- 수급자별 항목 합계가 담긴 xlsx 파일 다운로드
- 기존 시스템에 붙여넣기/업로드 가능
